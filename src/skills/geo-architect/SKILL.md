# GEO Architect Skill
Use this skill when analyzing page structure or generating content.

## Capabilities
1. **Schema Injection:** Automatically generates nested JSON-LD for `MedicalWebPage`, `InsuranceAgency`, and `GovernmentService`.
2. **Snippet Optimization:** Formats content to win "Position Zero" (Featured Snippets) and AI Overviews.
3. **Citation Analysis:** Cross-references content against high-authority sources (Medicare.gov, CMS.gov, IRS.gov) to ensure alignment.
4. **Entity Linking:** Identifies opportunities to semantically link internal pages based on overarching concepts rather than exact-match keywords.
